## 概述

参照canvas_flutter弹幕插件，使用LeafRenderObjectWidget替换 `CustomPainter`来实现，在canvas_flutter弹幕插件基础上支持点击悬停，弹幕按照进入时间层次渲染，因此动态弹幕和静态弹幕没有分离，弹幕量达到一定值时性能不如canvas_flutter。
